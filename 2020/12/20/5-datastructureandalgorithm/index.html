2.1 数组和链表

  1. 高级数据语言

    1. 对于数组的元素不做类型限制，称之为泛型

  2. 内存管理器

    为创建的数组元素分配对应的地址值

  3. 数组

    元素依靠索引相互关联

    1. 查询快O(1)
    2. 增删慢O(n)

  4. 单链表

    仅有一个前驱指针指向下一个元素

  5. 循环链表

    将单链表的最后一个指针指向头结点

  6. 双向链表(Java中的链表)

    每一个元素都有前驱指针和后驱指针来指向相邻元素

    1. 查询慢O(n)
    2. 增删快O(1)
    3. 应用于LRU Cache(最近最少使用)

  7. 小结数组链表的时间复杂度

    类型 数组 链表
    前追加 O(n)–>O(1) O(1)
    后追加 O(1) O(1)
    随机访问 O(1) O(n)
    添加 O(n) O(1)
    删除 O(n) O(1)

    1. 必须在原数组上操作，不能拷贝额外的数组。
    2. 尽量减少操作次数。

    实现：

    if(nums == null || nums.length == 0){;

  2. 数组(装水最多的容器)

    题述：给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0) 。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

    示例：

    输入：[1,8,6,2,5,4,8,3,7]
输出：49 
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

    提示：

    n = height.length
2 <= n <= 3 * 104
0 <= height[i] <= 3 * 104

    实现：

    // 枚举：left bar l & right bar r ：(r-l)*height_diff(这里的diff是差分)，时间复杂度是O(n^2)
int max = 0; // 初始化最大面积值
int area = 0; // 初始化当前面积值
// 外层循环遍历左边界，内循环遍历右边界
for (int i = 0; i < height.length; i++) {;
    for (int j = i+1; j < height.length; j++) {;
        // 选取左右边界的最小高作为当前矩形高，并计算左右边界差作为当前矩形长
        // 由此计算当前矩形面积
        area = (j-i)*Math.min(height[i],height[j]);
        // 始终将最大的面积值存放在max变量
        max = Math.max(max,area);
    };
};
// 返回最大面积
return max;

/**
 * @Description // 双向交叉计算矩形面积取最大值
 **/
// 初始最大面积值
int max = 0;
// 初始化右边界
int right = height.length - 1;
// 初始化左边界
int left = 0;
// 初始化当前面积
int area;
while(right != left){;
    // 如果右边界的高度值大于左边界
    if(height[right] > height[left]){;
        // 则当前面积为左边界的高*左右边界之差
        area = height[left] * (right - left);
        // 左边界向前走
        left += 1;
    };else{;
        // 否则当前面积为右边界的高*左右边界之差
        area = height[right] * (right - left);
        // 右边界向后走(以数组索引递增为正方向)
        right -= 1;
    };
    // 返回最大值的面积值
    max = Math.max(max, area);
};
// 返回最大面积
return max;

// 使用三元运算符简化最优解
// 初始最大面积值
int max = 0;
// 初始化右边界
int right = height.length - 1;
// 初始化左边界
int left = 0;
// 初始化当前面积
int area;
while (right != left) {;
    // 计算最小高度，注意这里返回的是left+1和right-1处的高度值
    int minHeight = height[right] > height[left] ? height[left++] : height[right--];
    // 这里+1是补加left和left+1间的矩形面积，同样也是补加right和right-1之间的面积值(谁小补谁)
    area = minHeight * (right - left + 1);
    max = Math.max(max, area);
};
// 返回最大面积
return max;
